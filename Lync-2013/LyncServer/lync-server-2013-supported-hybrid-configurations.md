Weitere Informationen zum Client Support finden Sie unter [Client Vergleichstabellen für lync Server 2013](lync-server-2013-desktop-client-comparison-tables.md) und Skype for Business Online-Vergleichstabellen für [Clients für Skype for Business Online](https://go.microsoft.com/fwlink/p/?linkid=281902).
