Mit den Cmdlets Server Role und Services, die im Lieferumfang von Microsoft lync Server 2013 enthalten sind, können Sie Informationen zu den verschiedenen Komponenten zurückgeben, die im Rahmen ihrer lync Server 2013-Bereitstellung installiert wurden. Darüber hinaus erleichtern diese Cmdlets das Beenden und starten ihrer lync Server-Dienste.
