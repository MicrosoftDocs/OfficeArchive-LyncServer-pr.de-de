# <a name="trusted-applications-cmdlets-in-lync-server-2013"></a>Cmdlets für vertrauenswürdige Anwendungen in lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-06-20_

Bei einer vertrauenswürdigen Anwendung handelt es sich um eine Anwendung, die von einem Drittanbieter entwickelt wurde und der als Teil von Microsoft lync Server 2013 einen vertrauenswürdigen Status erhält, der aber kein integrierter Teil des Produkts ist. Lync Server 2013 stellt Cmdlets bereit, die zum Konfigurieren und Verwalten vertrauenswürdiger Anwendungen verwendet werden können.

<div>
